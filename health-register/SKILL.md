# 医院挂号

根据用户最新输入，结合当前日期、必要的上下文与用户记忆，提取一个完整、明确的医院挂号参数集合。



## 核心执行规则（必须严格遵循）

### 1. 需求分析
- 是否调用技能，只基于**用户最新输入**判断。
- 当前问题属于**医院挂号操作**时，调用该技能。
- 仅提取用户**明确说明**的信息；未明确提及的参数不得补充。

### 2. 上下文与记忆融合
- **上下文**仅用于补全省略、指代、续问。
- **记忆**仅用于补充与当前挂号直接相关的稳定背景，如长期就诊医院、常挂科室等；若当前输入未明确涉及，不得擅自补入。
- 信息优先级：**用户最新输入 > 当前上下文 > 用户记忆**。
- 仅在确有依据时补充；有冲突时以**用户最新输入**为准。

### 3. 时间提取规则
- `startTime` 与 `endTime` **必须成对出现**，要么同时提取，要么都不提取。
- 时间提取必须能确定**日期 + 时间范围**；否则不提取。
- 日期可来自以下四类信息：
  1. 用户明确给出的绝对日期，如“2026-04-20”“4月20日”；
  2. 用户明确给出的相对日期，如“今天”“明天”“后天”“本周五”，需结合**当前日期**换算为具体日期；
  3. 当前上下文中已经明确给出的日期信息。
  4. 若只有“上午/下午/晚上/8点”等时间表达，默认按照**当前日期**提取。
- 若用户明确给出某个时间段，可按下列规则生成时间范围：
  - 上午 → `07:00` 到 `12:00`
  - 中午 → `10:00` 到 `14:00`
  - 下午 → `12:00` 到 `18:00`
  - 晚上 → `18:00` 到 `24:00`
  - 全天 → `00:00` 到 `24:00`
- 若用户明确给出单点时间，如“8点的号”“下午3点左右”，则以该时间为中心，前后各扩 2 小时生成时间范围；超出边界时按当天 `00:00` 和 `24:00` 截断。
- 若用户未明确说明时间，且上下文中也没有可确定的日期或时间信息，则 `startTime` 与 `endTime` 均不提取。

### 4. 安全与表达要求
- 信息不足时，仅保留已知事实，不编造医院、科室、医生、时间或号别。



## 输入参数

| 字段名 | 类型 | 必填 | 默认值 | 描述 |
|---|---|---|---|---|
| startTime | string | 否 | — | 挂号开始时间，格式为 `YYYY-MM-DD hh:mm`。 |
| endTime | string | 否 | — | 挂号结束时间，格式为 `YYYY-MM-DD hh:mm`。 |
| hospital | string | 否 | — | 挂号医院名称。 |
| department | string | 否 | — | 挂号科室名称。 |
| type | string | 否 | — | 号别。`1` 表示普通号，`2` 表示专家号。 |
| doctor | string | 否 | — | 挂号医生姓名。 |

> **调用前自检规则**：
> 1. 所有参数均为非必填，只有用户明确说明时才提取；
> 2. `startTime` 和 `endTime` 必须同时出现或同时不出现；
> 3. `startTime` 和 `endTime` 必须为 `YYYY-MM-DD hh:mm` 格式；
> 4. `type` 只能取值为 `1` 或 `2`；
> 5. 所有参数通过长选项传递，禁止使用 stdin 或 JSON；
> 6. 不得补造医院、科室、医生、时间或号别信息。



## 输出命令

示例：
```bash
cd <skill_root> && python scripts/python/health_register.py --hospital "协和医院" --department "心内科" --type "2"
```
```bash
cd <skill_root> && python scripts/python/health_register.py --startTime "2026-04-20 07:00" --endTime "2026-04-20 12:00" --hospital "北京大学第一医院" --department "皮肤科" --doctor "张三" 
```

>  说明：
> - `cd <skill_root> && python scripts/python/health_register.py` 为前面固定输出内容，后面携带参数；
> - `<skill_root>` 为技能根目录占位符，运行时替换；
> - 仅输出 bash 命令即可
